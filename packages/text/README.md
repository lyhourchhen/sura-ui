# Sura Text

> render text with bulit-in translation

# How to install

With NPM

```bash
npm i -D @@asurraa/sura-ui-text
```

With yarn

```bash
yarn add @asurraa/sura-ui-text
```

## Example

```tsx
import { AsurRaaText } from "@asurraa/sura-ui-text";

const Example = () => {
  return <AsurRaaText>Hello World</AsurRaaText>;
};
```
